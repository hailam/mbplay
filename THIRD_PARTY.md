# Third-Party Libraries

This project uses the following third-party libraries:

## CMT (C Metal)

- **Description:** C Bindings/Wrappers for Apple's Metal Graphics API
- **License:** MIT
- **URL:** https://github.com/recp/cmt

## QOI (Quite OK Image Format)

- **Description:** Fast, lossless image compression format
- **License:** MIT
- **URL:** https://github.com/phoboslab/qoi

## libpng

- **Description:** PNG reference library
- **License:** libpng license (BSD-style)
- **URL:** http://www.libpng.org/pub/png/libpng.html

## SDL2 (Simple DirectMedia Layer)

- **Description:** Cross-platform development library (used for CLI viewer)
- **License:** zlib License
- **URL:** https://www.libsdl.org/

## GMP (GNU Multiple Precision Arithmetic Library)

- **Description:** Arbitrary precision arithmetic
- **License:** LGPL v3 / GPL v2
- **URL:** https://gmplib.org/

## MPFR (GNU Multiple Precision Floating-Point Reliably)

- **Description:** Multiple-precision floating-point computations with correct rounding
- **License:** LGPL v3
- **URL:** https://www.mpfr.org/

---

All dependencies are managed through xmake and installed automatically during build.
